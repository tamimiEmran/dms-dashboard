Driver — binary

Is there a driver?

Predicted positive when ≥ 1 detection lands in the frame. Recall is the metric — Hikvision GT is 625 / 626 positives so accuracy saturates; misses are what matter.

Passenger count

How many besides the driver?

Predicted as max(0, n_detections − 1). MAE + exact-match + within-1 reported. Confusion matrix shows where each detector under- or over-counts.

Two detector families

Face vs. person boxes

Face counts miss occluded passengers. Person counts pick up driver arms / mirror reflections as bodies. Both rendered side-by-side.

Source. Predictions are cached detections.parquet from the face-person bootstrap experiments. No re-inference run for this evaluation — same boxes, new metric.

Methodology — What's Counted, What Isn't

Methodology — What's Counted, What Isn't

Inputs, metrics, caveats, and how to reproduce.

Inputs

Cached detection parquets

One detections.parquet per (detector, dataset). Schema: sample_id, model, class_name, x1, y1, x2, y2, conf. No filtering, no re-inference.

Metrics

Per-frame counts vs. integer GT

Driver: TP/FP/FN with positive = at least one detection. Passenger: MAE, RMSE, exact-match, within-1, count confusion matrix capped at 4.

Caveats

What this does not measure

Box localisation quality (use eval_vs_validation.json for IoU). Passenger ID consistency across frames. Driver-vs-passenger distinction beyond "subtract 1".

Reproduce

Two scripts

scripts/experiments/evaluate_occupancy_counts.py writes outputs/evaluation/occupancy_counts/results.json. build_occupancy_dashboard.py renders this HTML from that file.

Edge cases

The 1-frame outlier

Hikvision has one frame with driver_count = 0. Any non-zero detection there is a false positive on the driver-binary metric. Deliberate — the frame is real and the detector should not silently absolve itself.

Heat-map buckets

Absolute, not relative

Higher-better thresholds: 0.95 / 0.85 / 0.70 / 0.50. MAE thresholds: 0.20 / 0.40 / 0.60 / 0.80. Picked from observed range — not a baseline-relative scheme.
